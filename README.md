# Ctrl Alt Ship

> 中文读者：看 [简体中文版本](README.zh-CN.md)。

AI coding tools are moving fast. The hard part is still knowing what to try in a real engineering situation.

Ctrl Alt Ship is a scenario map for that moment. Start with the work in front of you, then open the matching scenario as a standalone guide.

## Start with your situation

| Situation | Start here |
| --- | --- |
| The request is fuzzy and needs to become workable tasks | [Requirements to Tasks](scenarios/requirements-to-tasks/README.md) |
| AI keeps missing project context or local conventions | [Project Context Memory](scenarios/project-context-memory/README.md) |
| You want AI help with implementation, migration, or refactoring | [Coding and Refactoring](scenarios/coding-and-refactoring/README.md) |
| You have a failure, but not a proven cause | [Debugging](scenarios/debugging/README.md) |
| AI changed the code and you need proof it still works | [Automated Verification](scenarios/automated-verification/README.md) |
| AI-generated work needs review before merge | [Code Review and Quality Gates](scenarios/code-review-quality-gates/README.md) |
| Decisions and project knowledge keep disappearing into chat | [Documentation and Knowledge](scenarios/documentation-knowledge/README.md) |
| A change needs release notes, rollout steps, or rollback thinking | [Release and Change Management](scenarios/release-change-management/README.md) |
| Logs, alerts, and chats need to become a clear incident timeline | [Incident Response](scenarios/incident-response/README.md) |
| A team needs shared AI usage boundaries | [Team AI Governance](scenarios/team-ai-governance/README.md) |

## How to use this repo

Pick the scenario that matches your current work. Each scenario page is a standalone detailed guide with decision guidance, mainstream solution paths, tool fit, a practical workflow, examples, verification checks, and common failure modes.

The tool notes are there for comparison after you know the scenario. They are not the starting point.

The [weekly radar](radar/README.md) tracks changes that may affect scenario recommendations.
